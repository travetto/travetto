# Scaffold Instructions
How to bootstrap a Travetto project with scaffold.

## Setup
1. Ensure node/npm environment is ready.
2. Configure basic git identity if prompted by workflow.
3. Run scaffold command via npx.

## Usage Workflow
- Start generator (`npx @travetto/scaffold`).
- Choose desired features/integrations.
- Review generated files, then install/run project.

Minimal pattern:
1. Execute scaffold.
2. Select web/model/auth/test features as needed.
3. Run generated app and tests.

## Safe Defaults
- Keep initial feature set minimal, then expand deliberately.
- Validate generated config before committing.
- Run tests immediately after generation.
