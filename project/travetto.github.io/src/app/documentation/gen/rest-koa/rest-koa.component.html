Rest-Koa

KOA Provider 
$ npm install @travetto/rest-koa

The module is a koa implementation of a RestApp. A valid customization of the RestApp would look like:

Customizing a Koa App 
export class SampleConfig {{ '{' }}

 @InjectableFactory()
  static customizer(): RestAppCustomizer<koa> {{ '{' }}
    return new (class extends RestAppCustomizer<koa> {{ '{' }}
      customize(raw: koa) {{ '{' }}
        raw.use(koaBunyanLogger());
    {{ '}' }})();
  {{ '}' }}
{{ '}' }}

Default Stack

When working with koa applications, the module provides what is assumed to be a sufficient set of basic filters. Specifically:

  • kCompress()
  • kBodyParser()
