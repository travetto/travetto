Rest-Aws-Lambda

AWS Provider 
$ npm install @travetto/rest-aws-lambda

The module is an aws-serverless-express provider for the Rest module.

Default Stack

When working with an express applications, the module provides what is assumed to be a sufficient set of basic filters. Specifically:

  • compression()
  • bodyParser.json()
  • bodyParser.urlencoded()
  • bodyParser.raw({{ '{' }} type: 'image/*' {{ '}' }})
