Module Structure
-----------------
## App Config
* (root)
  * doc/
  * src/
  * resources/
  * support/
  * test/
    * resources/
  * doc.ts
  * index.ts - Root index for loading

## Test
* test/
  * resources/

## Library (node_modules/@travetto/{x})
* bin/ - Pure JS files for execution
* support/ -
  - bin/ support for cli operations
  - phase.* Phase support 
  - transform.* AST transformer support
  - main.* Main entry points
  - cli.* CLI entry points
